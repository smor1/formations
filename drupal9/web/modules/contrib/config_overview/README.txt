INTRODUCTION
------------

Config Overview is intended to help to understand how the configuration
is handled when synchronizing. It shows if configuration objects are
overridden locally or if it is excluded from imports/exports.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/config_overview

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/search/config_overview


REQUIREMENTS
------------

No special requirements.
